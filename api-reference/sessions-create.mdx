---
title: Create Session
api: POST /api/sessions
description: Create a new session.
---

<Note>Requires authentication. Maximum 20 sessions per DJ.</Note>

## Request

<ParamField body="name" type="string" required>
  Display name for the session.
</ParamField>

### Example Request

```bash
curl -X POST https://crowdplay-api-i1xh.onrender.com/api/sessions \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"name": "Saturday Beach Party"}'
```

## Response

<ResponseField name="session" type="object">
  The created session object, including the generated `code`.
</ResponseField>

### Example Response

```json
{
  "_id": "64a1b2c3d4e5f6a7b8c9d0e2",
  "name": "Saturday Beach Party",
  "code": "X7K9P2",
  "isActive": true,
  "dj": "64a1b2c3d4e5f6a7b8c9d0e0",
  "queue": [],
  "createdAt": "2024-01-16T18:00:00.000Z"
}
```

### Error Responses

| Status | Description |
|--------|-------------|
| 400 | Session limit reached (maximum 20) |
| 400 | Missing session name |
| 401 | Not authenticated |
