---
title: Remove from Queue
api: DELETE /api/sessions/{id}/queue/{itemId}
description: Remove a specific song from the queue.
---

<Note>Requires authentication.</Note>

## Request

<ParamField path="id" type="string" required>
  The MongoDB document ID of the session.
</ParamField>

<ParamField path="itemId" type="string" required>
  The unique ID of the queue item to remove.
</ParamField>

### Example Request

```bash
curl -X DELETE https://crowdplay-api-i1xh.onrender.com/api/sessions/64a1b2c3d4e5f6a7b8c9d0e2/queue/item-uuid-1 \
  -H "Authorization: Bearer <token>"
```

## Response

Returns the updated session object.

## Side Effects

- The song is completely removed from the queue
- Remaining songs' positions are recalculated
- A `queue-updated` Socket.IO event is emitted

### Error Responses

| Status | Description |
|--------|-------------|
| 404 | Session or queue item not found |
| 401 | Not authenticated |
