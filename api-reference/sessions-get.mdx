---
title: Get Session
api: GET /api/sessions/{id}
description: Get a specific session with its full queue.
---

<Note>Requires authentication. DJ can only access their own sessions.</Note>

## Request

<ParamField path="id" type="string" required>
  The MongoDB document ID of the session.
</ParamField>

### Example Request

```bash
curl https://crowdplay-api-i1xh.onrender.com/api/sessions/64a1b2c3d4e5f6a7b8c9d0e2 \
  -H "Authorization: Bearer <token>"
```

## Response

Returns the full session object including the complete queue with all items and their statuses.
