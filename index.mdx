## Architecture

CrowdPlay consists of three parts:

| Component | Tech | URL |
|-----------|------|-----|
| **API / DJ Dashboard** | Express.js + TypeScript + EJS | [crowdplay-api-i1xh.onrender.com](https://crowdplay-api-i1xh.onrender.com) |
| **Fan Client** | React + Vite SPA | [crowdplay-app.github.io/fan](https://crowdplay-app.github.io/fan/) |
| **Homepage** | React + Vite SPA | [crowdplay-app.github.io/home](https://crowdplay-app.github.io/home/) |

All real-time communication happens over **Socket.IO** WebSockets. Song data comes from the **Deezer API**.
